# تقرير فحص مشروع موقع المطعم - Bistro Bliss

## المشاكل المكتشفة

### 1. مشاكل البنية والمسارات

#### 1.1 موقع index.html غير صحيح
- **المشكلة**: ملف `index.html` موجود في مجلد `html/` بدلاً من جذر المشروع
- **التأثير**: GitHub Pages لن يجد الصفحة الرئيسية تلقائياً
- **الحل**: نقل `index.html` إلى جذر المشروع

#### 1.2 المسارات النسبية غير صحيحة
- **المشكلة**: جميع الملفات تستخدم `../css/` و `../js/` و `../photo/` بدلاً من المسارات الصحيحة
- **التأثير**: روابط معطلة عند نقل الملفات أو نشرها على GitHub Pages
- **الحل**: تحديث جميع المسارات لتكون متوافقة مع الهيكل الجديد

#### 1.3 مجلد الصور باسم `photo` بدلاً من `images`
- **المشكلة**: اسم المجلد غير واضح وغير متوافق مع المعايير
- **الحل**: إعادة تسمية المجلد إلى `assets/images`

### 2. مشاكل HTML

#### 2.1 صور بدون مسارات صحيحة
- **الملفات المتأثرة**: `index.html`, `about.html`, `contact.html`, `menu.html`
- **المشكلة**: صور مفقودة أو مسارات معطلة (مثل `../0d0a4c75-1dbb-4731-8e35-802f9072f9be.png`)
- **الحل**: توحيد جميع المسارات

#### 2.2 عنصر img بدون src
- **الملف**: `index.html` السطر 36
- **المشكلة**: `<img src=" " alt="">`
- **الحل**: إزالة العنصر أو إضافة صورة صحيحة

#### 2.3 أخطاء في البنية الدلالية
- **المشكلة**: استخدام `<p>` داخل `<a>` في الـ footer
- **الحل**: استخدام البنية الدلالية الصحيحة

#### 2.4 نقص في alt attributes
- **المشكلة**: صور بدون وصف (alt)
- **الحل**: إضافة وصف مناسب لكل صورة

#### 2.5 عدم وجود required attributes
- **المشكلة**: حقول form بدون `required` attribute
- **الحل**: إضافة validation attributes

### 3. مشاكل CSS

#### 3.1 أسماء ملفات غير صحيحة
- **المشكلة**: `sign up .css` (يحتوي على مسافات)
- **الحل**: إعادة تسمية إلى `signup.css`

#### 3.2 أكواد مكررة
- **المشكلة**: نفس الأنماط في ملفات متعددة
- **الحل**: دمج الأنماط المشتركة

#### 3.3 أنماط inline غير ضرورية
- **المشكلة**: استخدام `style` attribute في عدة أماكن
- **الحل**: نقل إلى ملفات CSS

### 4. مشاكل JavaScript

#### 4.1 مسارات الصور غير صحيحة
- **الملف**: `produ.js`
- **المشكلة**: جميع الصور تشير إلى `../image/` بدلاً من `../assets/images/`
- **الحل**: تحديث جميع المسارات

#### 4.2 عدم توافق الفئات
- **المشكلة**: في `produ.js` الفئات هي `"food"` لكن في `menu.html` تتوقع `"Breakfast"`, `"Main Dishes"`, إلخ
- **الحل**: توحيد الفئات

#### 4.3 أخطاء في البنية
- **الملف**: `produ.js` السطر 23
- **المشكلة**: `<div class="disbg""></div>` (علامة إغلاق مكررة)
- **الحل**: إصلاح البنية

#### 4.4 ملفات مفقودة
- **المشكلة**: `../assets/js/color-modes.js` و `../assets/dist/css/bootstrap.min.css`
- **الحل**: إزالة المراجع غير الضرورية

### 5. مشاكل checkout.html

#### 5.1 صفحة معطلة تماماً
- **المشكلة**: تحتوي على مراجع لملفات Bootstrap الخاطئة وتصميم من Bootstrap docs

#### 5.2 بيانات وهمية
- **المشكلة**: تحتوي على بيانات hard-coded بدلاً من استخدام بيانات السلة
- **الحل**: ربط الصفحة بـ cart.js

### 6. مشاكل GitHub Pages

### 7. مشاكل الأداء

#### 7.1 عدم وجود تحسين للصور
